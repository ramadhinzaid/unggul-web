# Unggul Web

Aplikasi web untuk manajemen penjualan, stok, dan pelanggan. Dibangun menggunakan React, TypeScript, dan Vite.

## Cara Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/ramadhinzaid/unggul-web.git
cd unggul-web
```

### 2. Install Dependencies

Pastikan Anda memiliki [Node.js](https://nodejs.org/) terinstall. Kemudian, jalankan perintah berikut untuk menginstall semua dependency yang dibutuhkan.

```bash
npm install
```

### 3. Jalankan Aplikasi

Setelah instalasi selesai, jalankan aplikasi dalam mode development.

```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:5173` (atau port lain jika 5173 sudah digunakan).
